# Hotel Reservation System

A Hotel Reservation System developed in Java using Swing GUI. This application allows users to book hotel rooms, calculate the total bill based on the selected room type and number of days, and view booking details.

## Features

* Enter customer name
* Enter number of booking days
* Select room type:

  * Single Room (Rs. 2000/day)
  * Double Room (Rs. 4000/day)
  * Suite Room (Rs. 7000/day)
* Automatic bill calculation
* Display booking details
* Reset form functionality
* User-friendly Java Swing interface

## Technologies Used

* Java
* Swing (GUI)
* AWT Event Handling
* Object-Oriented Programming (OOP)

## How to Run

1. Clone the repository:
   git clone https://github.com/ali-mehdi-langah/java-hotel-reservation-system.git

2. Open the project in your Java IDE.

3. Compile and run:
   HotelReservationSystem.java

## Output

The application displays:

* Customer Name
* Number of Days
* Selected Room Type
* Total Bill Amount
